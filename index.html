Curso Técnico em Desenvolvimento de Sistemas

Resumo

O Curso Técnico de Desenvolvimento de Sistemas tem por objetivo habilitar profissionais para analisar requisitos funcionais e não funcionais de produtos, desenvolver e testar sistemas de software, de acordo com as especificações do projeto, considerando as boas práticas do mercado de tecnologia da informação e as necessidades do usuário.

CARGA HORÁRIA
1200 horas

Programação do Curso

HARDWARE E REDES: 62 horas
  • Hardware; Redes de computadores.

    • SISTEMAS OPERACIONAIS: 62 horas
  • Arquitetura dos sistemas operacionais; Sistema operacional código fechado; Sistemas operacional código aberto; Editor de texto; Planilha eletrônica.

    • LÓGICA DE PROGRAMAÇÃO: 40 HORAS
  • Lógica de Programação e Algoritmos; Software; Programação; Gestão do Tempo

    • VERSIONAMENTO: 16 horas
  • Metodologias de versionamento.

    • CODIFICAÇÃO PARA FRONT-END: 140 horas
  • Técnicas de levantamento de requisitos; User Experience (UX); User Interface (UI); Prototipagem; Linguagem de marcação; CSS; Frameworks; Linguagens de programação.

    • INTERAÇÃO COM APIs: 40 horas
  • Segurança da informação; Serviços server-side; Estrutura organizacional.

    • TESTE DE FRONT-END: 40 horas
  • Processo fundamental de teste; Planejamento de testes client-side; Conceitos fundamentais; Tipos de testes; Técnicas de testes; Automação de testes.

    • CODIFICAÇÃO BACK-END: 164 horas
  • Metodologias de desenvolvimento; Segurança da informação; Validação de sistemas; Implantação de sistemas; Melhoria contínua.

    • BANCO DE DADOS: 36 horas
  • Documentação técnica; Modelo entidade-relacionamento; Banco de dados; Linguagem de consulta de banco de dados; Segurança da informação.

  • API; Métodos de requisição HTTP; Padrão Model View Control (MVC); Linguagem de programação para APIs; Relações Institucionais verticais e horizontais.

    • TESTES DE BACK-END: 62 horas
  • Testes server-side; Processo fundamental de teste; Tipos de testes; Níveis de testes; Frameworks de teste em sistemas server-side; Práticas de testes.

    • BANCO DE DADOS EM APLICATIVOS MÓVEIS: 90 horas
  • Sistemas operacionais; Aplicativos; Recursos embarcados; Resolução de problemas.

    • CODIFICAÇÃO E TESTES DE APLICATIVOS MOBILE: 162 horas
  • Dispositivos móveis; Criação de interface; Recursos de hardware; APIs; Persistência de dados em dispositivos móveis; Consumo de RESTfull web servisse; Publicação do aplicativo.

    • METODOLOGIAS ÁGEIS: 12 horas
  • Requisitos do projeto; Manifesto Agile; Scrum; Autogestão.

    • PROJETOS: 214 horas
  • Qualidade de software; Metodologias de desenvolvimento; Metodologia de gerenciamento de projeto; Apresentação do projeto.
